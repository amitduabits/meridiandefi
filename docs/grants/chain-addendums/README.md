## Priority Rationale

### HIGH Priority

- **NEAR:** $20M fund dedicated specifically to AI agents. Maximum strategic alignment -- NEAR's entire ecosystem thesis maps to Meridian's architecture. Account model uniquely suited for agent identity. Apply immediately.
- **Starknet:** Up to $1M per grant. Cairo compilation path is a genuine technical differentiator -- provably correct agent execution is a category-defining capability. Largest individual grant potential in the pipeline.

### MEDIUM Priority

All other chains offer strong ecosystem fit and meaningful funding, but lack the deep strategic alignment (NEAR) or outsized funding potential (Starknet) of the high-priority targets. Apply in parallel with high-priority applications.
