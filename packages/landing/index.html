Process

From idea to autonomous agent in minutes

Meridian handles the complexity. You define the strategy ‚Äî the framework handles execution, risk, and coordination.

01
Define Strategy

Use natural language, DSL, or TypeScript to describe what your agent should do.

// Natural language ‚Üí strategy const strategy = await translateStrategy("Rebalance portfolio when ETH drifts >5% from target. Use Uniswap V3, max 0.3% slippage."); // Or use the built-in presets const s = createRebalancerStrategy({ driftThreshold: 0.05, targetAllocations: { ETH: 0.4, USDC: 0.3 } });
02
Deploy Agent

Agents run autonomously with Sense ‚Üí Think ‚Üí Act ‚Üí Reflect decision cycles.

const agent = new Agent({ id: "rebalancer-001", chains: [421614], // Arbitrum tickIntervalMs: 30_000, dryRun: false, }, { eventBus, sense, think, act }); agent.setStrategy(strategy); await agent.start(); // Sense ‚Üí Think ‚Üí Act ‚Üí Reflect // loops automatically every 30s
03
Monitor & Scale

Real-time dashboard, circuit breakers, multi-agent collaboration out of the box.

// Subscribe to agent events eventBus.on("agent:decision", (e) => { console.log(e.record.reasoning); }); // Multi-agent team const analyst = new MarketAnalyst(bus); const risk = new RiskManager(); const exec = new Executor(); // Analyst ‚Üí Risk ‚Üí Executor pipeline
Capabilities

Everything DeFi needs. Nothing it doesn't.

Built from the ground up for financial autonomy ‚Äî not retrofitted from a general-purpose agent framework.

üß†
DeFi-Native Reasoning

Not a chatbot with a wallet. Meridian agents understand liquidity depth, slippage, impermanent loss, and gas optimization natively. Claude Sonnet handles trade reasoning; local LLMs handle real-time signals.

‚õìÔ∏è
Cross-Chain Execution

One agent, many chains. Unified connectors for Ethereum, Arbitrum, Base, Optimism, Polygon, Solana, and more. Protocol adapters for Uniswap V3, Aave V3, Curve, Lido, Jupiter, and growing.

ü§ù
Agent Collaboration

Agents discover, negotiate, and pay each other via libp2p GossipSub and on-chain escrow. An Analyst hires a Risk Manager, which delegates to an Executor ‚Äî all trustless and verifiable.

üõ°Ô∏è
Risk Management

Pre-flight validator checks every transaction: position size, gas cost, slippage, contract approval, daily loss limits, and simulation. Circuit breakers halt activity on anomalies automatically.

üìê
Strategy DSL

Write strategies in natural language, a typed DSL, or pure TypeScript. The compiler validates, sandboxes in isolated-vm, and backtests against historical DuckDB data before deployment.

üìä
Monitoring Dashboard

React 18 dashboard with real-time agent state, portfolio P&L, transaction history, risk metrics, and strategy performance. tRPC API, WebSocket updates, SIWE authentication.
